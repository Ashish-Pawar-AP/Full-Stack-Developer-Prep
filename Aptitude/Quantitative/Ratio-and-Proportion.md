---

## 📌 Important Concepts

- Ratio = a : b = a/b  
- If a : b = c : d → a/b = c/d (Proportion)  
- If a : b = m : n → a = km, b = kn  
- Compound ratio → (a:b) × (c:d) = ac : bd  
- Inverse proportion → If one increases, other decreases
