## 📌 Important Formulae

- sinθ = Opposite / Hypotenuse  
- cosθ = Adjacent / Hypotenuse  
- tanθ = Opposite / Adjacent  
- tanθ = sinθ / cosθ  
- sin²θ + cos²θ = 1  
- 1 + tan²θ = sec²θ  
- 1 + cot²θ = cosec²θ  

### Standard Values

| θ | 0° | 30° | 45° | 60° | 90° |
|----|----|-----|-----|-----|-----|
| sinθ | 0 | 1/2 | 1/√2 | √3/2 | 1 |
| cosθ | 1 | √3/2 | 1/√2 | 1/2 | 0 |
| tanθ | 0 | 1/√3 | 1 | √3 | ∞ |

**Answer:**

## Q32.
If sinθ + cosθ = 1, find sinθcosθ.

**Answer:**  
Square both sides  

sin²θ + cos²θ + 2sinθcosθ = 1  

1 + 2sinθcosθ = 1  

sinθcosθ = 0
