# 🗣️ Aptitude Verbal – Active & Passive Voice  
(Level-wise: Easy, Medium, Hard, Very Hard)  
Each section contains practice questions with answers.

---

## 📌 Basic Rule

**Active Voice Structure:**  
Subject + Verb + Object  

**Passive Voice Structure:**  
Object + Helping Verb (be form) + V3 + by + Subject  

Example:  
Active → Ram wrote a letter.  
Passive → A letter was written by Ram.
