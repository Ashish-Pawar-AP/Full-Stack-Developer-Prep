# 🧠 Logical Reasoning – Cause & Effect  
(Level-wise: Easy, Medium, Hard, Very Hard)  
Each section contains 10 Questions with Answers.

---

## 📌 Basic Concept

- **Cause** → Reason behind something happening  
- **Effect** → Result of that cause  
- Sometimes:
  - One cause → Multiple effects  
  - Multiple causes → One effect  
  - Independent events (No relation)  

---

# 🟡 MEDIUM LEVEL (11–20)

For each question, choose:

1️⃣ Statement I is cause, II is effect  
2️⃣ Statement II is cause, I is effect  
3️⃣ Both independent  
4️⃣ Both are effects of common cause  

---
