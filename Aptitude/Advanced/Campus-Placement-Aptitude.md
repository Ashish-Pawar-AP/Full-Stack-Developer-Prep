# 🟢 3️⃣ Quant Strategy (Campus Focus)

### 🔹 Focus Topics (High Frequency)
✔ Percentages  
✔ Ratio  
✔ Averages  
✔ Time & Work  
✔ Speed & Distance  

### 🔹 Must Memorize

- Cubes till 15  
- Fraction–Percentage conversion  
- Divisibility rules  

### 🔹 Speed Trick Rule
👉 If question looks lengthy → Try option substitution
