# 🟢 1️⃣ Percentages

Percentage = (Value / Total) × 100  

Increase % = (Increase / Original) × 100  

Decrease % = (Decrease / Original) × 100  

New Value after x% increase = Original × (1 + x/100)  

New Value after x% decrease = Original × (1 − x/100)  

Successive % change = A + B + (AB/100)

If increased by x% then decreased by x% → Net change = − x² /100  

---

# 🟢 2️⃣ Profit & Loss

Profit = SP − CP  
Loss = CP − SP  

Profit % = (Profit / CP) × 100  
Loss % = (Loss / CP) × 100  

SP = CP × (1 + x/100)  
SP = CP × (1 − x/100)  

Discount % = (Discount / Marked Price) × 100  

Successive discount = A + B − (AB/100)

---

# 🟢 3️⃣ Ratio & Proportion

If A : B = m : n  
Then A = mk, B = nk  

If a/b = c/d → ad = bc  

Fourth proportional = (b × c)/a  

Mean proportional = √(ab)

---

# 🟢 4️⃣ Average

Average = Sum of observations / Number  

New average method:  
Total change = Difference × Total items  

If equally spaced numbers →  
Average = Middle number  

---

# 🟢 5️⃣ Time & Work

Work = Rate × Time  

If A does work in x days → 1 day work = 1/x  

If A & B together →  
Total efficiency = 1/x + 1/y  

Time = 1 / (Total efficiency)

Men & Days → Inverse proportion  

---

# 🟢 6️⃣ Time, Speed & Distance

Speed = Distance / Time  
Distance = Speed × Time  
Time = Distance / Speed  

Relative speed:  
Opposite direction → Add  
Same direction → Subtract  

Average speed (equal distance) = (2ab)/(a + b)

---

# 🟢 7️⃣ Boats & Streams

Downstream speed = B + S  
Upstream speed = B − S  

Boat speed = (Down + Up)/2  
Stream speed = (Down − Up)/2  

---

# 🟢 8️⃣ Simple Interest

SI = (P × R × T) / 100  

Amount = P + SI  

---

# 🟢 9️⃣ Compound Interest

Amount = P(1 + R/100)^T  

CI = Amount − Principal  

For 2 years:  
CI − SI = P(R²)/10000  

---

# 🟢 🔟 Mixtures & Alligation

Ratio = (Difference cross method)

Mean price = (a × x + b × y) / (x + y)

---

# 🟢 1️⃣1️⃣ LCM & HCF

LCM × HCF = Product of numbers  

HCF → Division method  

---

# 🟢 1️⃣2️⃣ Number System

Sum of first n natural numbers = n(n+1)/2  

Sum of first n squares = n(n+1)(2n+1)/6  

Sum of first n cubes = [n(n+1)/2]²  

---

# 🟢 1️⃣3️⃣ Permutation & Combination

nPr = n! / (n−r)!  

nCr = n! / [r!(n−r)!]  

Circular arrangement = (n−1)!  

---

# 🟢 1️⃣4️⃣ Probability

Probability = Favorable outcomes / Total outcomes  

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)  

If independent:  
P(A ∩ B) = P(A) × P(B)  

---

# 🟢 1️⃣5️⃣ Trains

Time to cross pole = Length / Speed  

Time to cross platform = (Train + Platform)/Speed  

---

# 🟢 1️⃣6️⃣ Pipes & Cisterns

Pipe efficiency = 1/time  

Total efficiency = Sum of efficiencies  

Leak → Negative efficiency  

---

# 🟢 1️⃣7️⃣ Heights & Distances

tanθ = Opposite / Adjacent  

sinθ = Opposite / Hypotenuse  

cosθ = Adjacent / Hypotenuse  

---

# 🟢 1️⃣8️⃣ Quadratic Equation

ax² + bx + c = 0  

x = [-b ± √(b² − 4ac)] / 2a  

Nature of roots:  
b² − 4ac > 0 → Real  
= 0 → Equal  
< 0 → Imaginary
